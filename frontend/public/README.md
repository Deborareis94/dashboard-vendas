
# Dashboard de Gestão de Vendas e Estoque

Aplicação Web Full Stack para gerenciamento de produtos, vendas e estoque.

## Funcionalidades

- Desenvolvimento de dashboard em React para gerenciamento de produtos e vendas.
- Implementação de CRUD de produtos com controle de estoque.
- Registro de vendas com atualização automática do estoque.
- Cálculo de faturamento diário com filtro por data.
- Desenvolvimento de API REST com Node.js e Express.
- Integração com banco de dados MySQL utilizando Sequelize.
- Interface moderna e responsiva com Tailwind CSS.

## Tecnologias

-  Frontend React, JavaScript, Tailwind CSS  
- Backend: Node.js, Express, API REST  
- Banco de dados: MySQL, Sequelize
